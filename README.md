# Tasks Tracker V1 (черновик)

Минимальная версия трекера задач, только чтобы **что-то работало**.

## Как запустить

- Откройте файл `index.html` двойным кликом или через браузер (File → Open File).

## Что умеет

- Добавлять задачу через поле ввода и кнопку **«Добавить»**
- Показывать живой список задач
- Отмечать задачу как выполненную / снимать отметку
- Удалять задачу из списка
